1. Download semua file yg ada di github
2. Masukin semua file aplikasi kedalam satu folder
3. Buka folder aplikasi di VsCode
4. Buka Terminal terus tambahin git bash 
5. Buat lingkungan virtual environment dengan memasukkan kode ini di git bash : python -m venv venv
6. Aktifin environmentnya : source venv/Scripts/activate
7. install semua library yg dibutuhin : pip install -r requirements.txt
